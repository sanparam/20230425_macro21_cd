0. 주의 사항
Macro21 E-Catalog CD에 제공되는 모든 정보는 임의로 재사용될 수 없읍니다. 
임의로 재사용시에는 법적인 처벌을 받을 수 있습니다. 또한 제공된 정보는 
귀사(귀하)의 내부 자료로만 이용해야 하며, 외부에 제공하거나 누설할시에 
발생할 수 있는 민, 형사상의 책임은 당사에서 일체 지지 않습니다. 
또한 정보의 내용은 이용일 현재의 내용과 다를 수 있습니다. 

저희 Macro21이 제공하는 정보를 이용한 귀사(귀하)의 판단이나 행위 결과에 
대해 당사에서는 일체 책임을 지지 않습니다. 

1. 설치(Install)
setup.exe를 실행시키면 Macro21 E-Catalog가 설치됩니다. 
기본적인 경로는 C:\Program Files\Macro21입니다.

2. 실행
반드시 Macro21 E-Catalog를 CD-ROM Drive에 넣고, Windows "시작"버튼에 있는 
"Macro21 E-Catalog"를 실행시키면 됩니다.

3. 삭제(Uninstall)
Windows "제어판"에서 "프로그램 추가/삭제"를 선택하신 후, Macro21을 삭제
하시면 됩니다.
