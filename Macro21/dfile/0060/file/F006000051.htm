#공구

공구 이용법

공구를 그래픽환경에서 이용가능하다.

 

공구 목록

공구 목록 안에서 공구순서 재 배열을 쉽게 하는 Graphic object interface . 공구번호 교환이 가공 중에 자동적으로 교환된다.

공구 자료실
 rough, finish, and ball end mills, drill, rigid tap, back bore, thread cutter, keyway cutters, fly cutters, counter sinks, shell, and face mills등이 평삭 공구에 포함된다.모든 종류의 앵글을 그리거나 코너 가공을 선명하게 할 수 있다

편리한 공구

간단한 형상의 정밀한 Rendering은 몇 가지 공구로도 뚜렷이 가공할 수 있다.
